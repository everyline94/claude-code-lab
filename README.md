# Claude Code Lab

Laboratório prático de aprendizado com Claude Code. Cada projeto resolve um problema real e ensina um conceito novo — de pipelines de transcrição a extratores de PDF, agentes autônomos e automações de produção.

---

## Projetos

| # | Projeto | Status | O que faz |
|---|---------|--------|-----------|
| 01 | [Transcriber](./Projeto%2001%20-%20Transcriber/) | ✅ Concluído | Recebe URL ou arquivo de vídeo e devolve transcrição limpa ou anotações de estudo em Markdown |
| 02 | [PDF Extractor](./Projeto%2002%20-%20PDF%20Extractor/) | ✅ Concluído | Combina descrição de aula + PDF de mapa mental e gera documento de estudo estruturado em Markdown |
| 03 | Renomeador de Assets | 🔜 Em breve | Organiza pastas renomeando arquivos em lote com convenção de nomenclatura |
| 04 | Monitor de Referências | 🔜 Em breve | Monitora perfis e canais periodicamente e salva conteúdo novo automaticamente |
| 05 | Brief Parser | 🔜 Em breve | Estrutura briefings de cliente (texto, áudio ou e-mail) no formato do processo de entrega |
| 06 | Gerador de Análise Estratégica | 🔜 Em breve | Pesquisa na web e gera documento de análise estratégica de cliente ou mercado |
| 07 | Pipeline de Entrega | 🔜 Em breve | Pipeline completo de entrega: checklist, organização de arquivos e histórico |

---

## Ferramentas usadas

| Ferramenta | Uso |
|------------|-----|
| Claude Code + Antigravity | Interface principal para rodar os agentes |
| yt-dlp | Download de áudio a partir de URLs |
| ffmpeg | Conversão e processamento de áudio |
| OpenAI Whisper | Transcrição de fala para texto |
| pdfplumber | Extração de texto de arquivos PDF |
| Python 3 | Runtime dos scripts |
